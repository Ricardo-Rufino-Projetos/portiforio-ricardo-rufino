# 🔍 Sistema de Auditoria Inteligente de Dados de Vendas

## 📝 Descrição do Projeto
Este projeto consiste em um sistema de auditoria automatizada desenvolvido em Python para monitorar, analisar e validar registros de vendas por meio de regras de segurança operacional, cálculo estatístico e detecção de inconsistências.

O principal objetivo é identificar discrepâncias financeiras, prevenir falhas humanas e reforçar a confiabilidade dos dados inseridos, utilizando lógica condicional para verificar padrões anormais, limites de segurança e necessidade de revisão manual.

Desenvolvido como projeto acadêmico e prático de **Lógica de Programação, Segurança de Dados e Automação**, o sistema realiza análises em tempo real sobre entradas fornecidas pelo usuário, simulando processos básicos de auditoria corporativa e controle interno.

## 🚀 Tecnologias Utilizadas
* **Linguagem:** Python 3
* **Recursos:** Estruturas condicionais, funções, loops de repetição
* **Ferramentas:** Terminal / Console, IDE Python
* **Métodos:** Cálculo de média, validação de limite, análise de discrepância

## 📊 Resultados e Aprendizados
O projeto demonstrou eficiência na automação de verificações básicas de integridade de dados.

* **Detecção automatizada de discrepâncias** em vendas fora do padrão esperado.
* **Sistema de quarentena** para médias acima do limite de segurança definido.
* **Validação manual assistida** para registros suspeitos.
* **Aprimoramento em lógica procedural**, controle de fluxo e segurança operacional.
* **Base sólida para futuras expansões**, como dashboards, banco de dados e machine learning.

## ⚙️ Funcionalidades Principais
* Cálculo automático da média de vendas
* Monitoramento de limite de segurança
* Identificação de vendas anormais
* Revisão manual de inconsistências
* Ajuste dinâmico de parâmetros de segurança

## 📂 Estrutura do Processo
```text
1. Definir limite de segurança
2. Inserir vendas
3. Calcular média
4. Validar quarentena
5. Detectar discrepâncias
6. Revisar valores suspeitos
7. Exibir relatório final
```
